# 🧾 Garage Billing Management System

A command-line based Garage Billing Management System developed using Java. The application helps manage customer records, vehicle services, and invoice generation efficiently through a menu-driven interface.

---

## 🚀 Features

- Add customer and vehicle details
- Manage garage services
- Generate invoices automatically
- Calculate total billing amount
- Menu-driven CLI interface
- Modular Java code structure

---

## 🛠️ Tech Stack

- Java
- OOPs
- Collections Framework
- Command Line Interface (CLI)

---

## 📂 Project Structure

```bash
src/
 ├── Car.java
 ├── Customer.java
 ├── GarageMain.java
 ├── GarageService.java
 ├── Invoice.java
 └── Service.java
```

---

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Garage-Billing-Management-System.git
```

2. Open project in IntelliJ IDEA / VS Code

3. Compile and run:

```bash
javac GarageMain.java
java GarageMain
```
